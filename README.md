# What is Helix ?

***Preamble***<br>
&nbsp;&nbsp; *This is a personal project that doesn't goal to be commercialized or used by other people.*<br>
&nbsp;&nbsp; *It's just a fun project to me and my future creation.*


So, **Helix** is a C++ library which is a wrapper of the SDL2 library to simplify and help to create games.
Many functions and long algorithms that we had to write with the SDL2 are replace by short and simple functions.


To illustrate how **Helix** works and most of these features, I create templates project that used **Helix**.
The templates code are in the *'templates'* folder of the repository and the executable are in the *'bin'* folder.
